# Quiz de Segurança Corporativa

Projeto Spring Boot com Thymeleaf, Bootstrap 5 e banco H2 em memória.

O banco de perguntas é focado em segurança máxima para o setor administrativo, cobrindo dados, documentos, e-mail, sites e boas práticas corporativas.

## Como executar

Use `mvn spring-boot:run` ou o wrapper gerado com `mvnw spring-boot:run`.
